# Frontend VoxAccess

## Instalação

```bash
npm install
```

## Executar em Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`

## Build para Produção

```bash
npm run build
```

Os arquivos estarão em `dist/`

## Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Axios
